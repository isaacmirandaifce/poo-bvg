# Projeto 2 - POO

## Solução

O sistema foi reestruturado utilizando:

- Herança (classe Seguro)
- Polimorfismo (cada seguro calcula seu próprio valor)
- Encapsulamento (CPF protegido)
- Abstração (classe base abstrata)

O uso de if/else foi eliminado, deixando o código mais organizado e fácil de manter.

## Execução

python src/main.py
