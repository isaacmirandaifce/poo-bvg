# Projeto 5 – Sistema Acadêmico Ampliado (C++)

## Descrição:
Sistema acadêmico desenvolvido em C++ utilizando Programação Orientada a Objetos. O projeto aplica herança, polimorfismo, sobrecarga, sobrescrita e herança múltipla.

O sistema permite o cadastro de:
- Alunos
- Professores
- Funcionários Administrativos
- Monitores

Cada tipo de usuário possui um relatório específico.

---------------------------------------------------------------

## Conceitos Utilizados:
- Herança
- Herança múltipla (classe Monitor)
- Polimorfismo
- Sobrecarga
- Sobrescrita
- Modularização (.h e .cpp)
