# Projeto_5 - William Axel: Herança, Polimorfismo, Sobrecarga e Sobrescrita
Este projeto amplia o sistema de gerenciamento acadêmico, incluindo:
- Herança entre classes Usuario, Aluno, Professor, FuncionarioAdministrativo, Monitor
- Polimorfismo usando métodos virtuais e ponteiros/referências
- Sobrecarga de métodos na classe Aluno
- Sobrescrita de métodos gerarRelatorio() em todas as classes derivadas
- Modularização em arquivos separados (src/)
- Diagrama UML completo (docs/Projeto_5_UML.png)
## Como Compilar e Executar
```bash
cd src
g++ *.cpp -o ../projeto5
cd ..
./projeto5
```
