##  REQUISITOS PARA COMPILAÇÃO
Para a correta compilação e execução do projeto, são necessários:

- Compilador **g++** com suporte ao padrão **C++17** ou superior
- Ambiente Linux (Codespace, WSL ou similar)

Para verificar a instalação do compilador, utilize o comando:

```bash
g++ --version

Para compilar o sistema, utilize o comando:

g++ -std=c++17 *.cpp -o app

Após a compilação, execute o programa com o comando:

./app
