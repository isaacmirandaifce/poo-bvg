# Projeto Professor em C++

## Descrição
Este projeto tem como objetivo demonstrar a separação de código em C++ utilizando
arquivo de cabeçalho (.h) e arquivo de implementação (.cpp), aplicando conceitos
básicos de Orientação a Objetos.

O projeto trabalha com a classe `Professor`, responsável por armazenar e manipular
informações de um professor.

## Estrutura do Projeto
O projeto é composto por dois arquivos principais:

- `Professor.h`  
  Contém a declaração da classe `Professor`, seus atributos e métodos.

- `Professor.cpp`  
  Contém a implementação dos métodos declarados no arquivo `Professor.h`.

## Classe Professor
A classe `Professor` possui:

### Atributos
- `nome` (string): nome do professor  
- `matricula` (int): número de matrícula  
- `salario` (double): salário do professor  

### Métodos
- Construtor padrão  
- Construtor parametrizado  
- Métodos getters para acesso aos atributos  
- Método `setSalario` para alteração do salário  
- Método `exibirDados` para mostrar as informações do professor  

## Objetivo Educacional
Este projeto foi desenvolvido para fins acadêmicos, com o objetivo de:
- Compreender a separação entre declaração e implementação em C++
- Praticar encapsulamento
- Aplicar conceitos básicos de Orientação a Objetos

## Tecnologias Utilizadas
- Linguagem: C++
- Paradigma: Orientação a Objetos
